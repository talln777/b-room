# b-room サイト仕様・変更メモ

更新日: 2026-07-11

## 構成

- 静的サイト。ビルドは不要。
- エントリーポイント: `index.html`
- 基本スタイル: `css/reset.css`、`css/variables.css`、`css/style.css`、`css/responsive.css`
- デザイン微調整: `css/tuning.css`
- 画像アセット: `images/`

## ヒーロー画像

- 表示ファイルは `images/hero-mascot.png`。
- 元データは `images/hero-mascot-original.png` として保持する。
- 表示画像のキャンバスサイズは **333 × 344 px**。
- 透明背景を維持すること。白背景を合成しないこと。
- HTMLの `width` / `height` は画像の実寸と同じ `333` / `344` を指定し、高さは `auto` とする。画像の縦横比を強制変更しない。
- 光沢はCSSフィルターで控えめに調整している。形状・透明度を変える画像生成処理は行わない。

## ロゴ

- 使用画像: `images/logo.png`。
- ヘッダーとフッターでは、明るさ・彩度を少し抑えるフィルターを適用している。
- ロゴをマスク化して色を置き換える方式は、一部ブラウザで表示されないため使わない。

## タイポグラフィとレイアウト

`css/tuning.css` に参考デザインに合わせた調整を集約している。

- ヒーロー見出し: `clamp(34px, 3.2vw, 44px)`、行間 `1.42`
- ヒーロー本文: 16px、行間 `2`
- セクション見出し: 30px
- アプリ名: 32px（モバイル 29px）
- アプリカード: 最小高さ 458px、内側余白 `32px 32px 22px`
- モバイル（600px以下）はヒーロー見出し34px、カードの最小高さを解除する。

## バリューセクションのアイコン

- 絵文字ではなく、インラインSVGを使用する。
- 色は `currentColor`（`--green-dark`）の単色線画。
- 1つ目「シンプルで使いやすい」: スライダーコントロールのアイコン。
- 2つ目「日常をもっと楽しく」: ハート。
- 3つ目「安心・安全」: 盾。線幅は 3.5。
- 4つ目「つながる体験を」: 2人組のアイコン。線幅は 1.75。

## フッターSNS

- Xは文字グリフ `𝕏`。
- Instagramはカメラ枠・レンズ・フラッシュ点を含むインラインSVG。
- URLは未設定のため、リンク先の `href="#" ` は公開前に実URLへ変更する。

## 公開前チェック

- `href="#" ` のリンク先を実URLへ変更する。
- hero・アプリアイコン・ロゴが存在することを確認する。
- PC／モバイルでヒーロー画像が縦横比を維持していることを確認する。
- インラインSVGアイコンとInstagramアイコンの表示を確認する。
