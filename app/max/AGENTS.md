# AGENTS

## Project

mAx Landing Page

## 最重要ルール

デザインモック画像が仕様です。

### 禁止事項

-   レイアウト変更
-   余白変更
-   色変更
-   フォント変更
-   サイズ変更
-   画像変更
-   セクション追加
-   セクション削除（ユーザー指示を除く）
-   勝手な改善

### 許可

-   HTML化
-   CSS化
-   レスポンシブ対応
-   アクセシビリティ改善
-   軽微なコード最適化

## 技術

-   HTML5
-   Vanilla CSS
-   Vanilla JavaScript
-   GitHub Pages
-   CSS Variables
-   Semantic HTML

## Goal

モック画像を忠実にHTML/CSSへ変換すること。
